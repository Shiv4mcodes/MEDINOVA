# 📖 About MEDINOVA

MEDINOVA is a modern healthcare dashboard designed to simplify medication management while promoting healthier daily habits.

The project brings together medicine scheduling, AI-assisted prescription scanning, medication safety checking, wellness tracking, adherence analytics, and accessibility features inside a single user-friendly interface.

Rather than acting as just another medicine reminder, MEDINOVA aims to provide users with a complete digital healthcare companion.

# 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/Shiv4mcodes/medinova.git
```

Move into the project

```bash
cd medinova
```

Install dependencies

```bash
npm install
```

Start development server

```bash
npm start
```

Create production build

```bash
npm run build
```
